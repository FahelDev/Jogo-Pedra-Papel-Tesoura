# Jogo-Pedra-Papel-Tesoura
Projeto simples desenvolvido em Java com o objetivo de praticar conceitos básicos de programação, lógica condicional, entrada de dados e geração de números aleatórios.

O jogo funciona no terminal/console e permite que o usuário jogue Pedra, Papel e Tesoura contra o computador. 🪨📄✂️

🚀 Funcionalidades

- ✅ Escolha do jogador via terminal
- ✅ Geração automática da jogada do computador
- ✅ Sistema de validação de entrada
- ✅ Verificação de vitória, derrota ou empate
- ✅ Exibição das escolhas do jogador e do computador

🛠️ Tecnologias Utilizadas
- Java
- Scanner (java.util.Scanner)
- Random (java.util.Random)
📚 Conceitos Praticados

Durante o desenvolvimento deste projeto, foram aplicados conceitos importantes de programação, como:

- Estruturas condicionais (if e else)
- Métodos/Funções em Java
- Entrada de dados com Scanner
- Geração de números aleatórios
- Organização de código
- Lógica de jogos simples
  
# 🎯 Como Jogar

O jogo funciona através de números:

Número	Escolha
- 0	Pedra
- 1	Papel
- 2	Tesoura
